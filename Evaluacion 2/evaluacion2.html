📘 Evaluación 2 - Clonar un blog con JavaScript Vanilla

Asignatura: Programación Front End  |  Modalidad: Individual  |  Nota máxima: 7,0

Docente: Lautaro Fernando Cornejo Arcos  |  Email: lautaro.cornejo@inacapmail.cl

⚠️ Importante: Esta evaluación se debe resolver exclusivamente con JavaScript Vanilla (puro). No se permite el uso de librerías ni frameworks externos (jQuery, React, Vue, Angular, Bootstrap JS, etc.).

🎯 Objetivos de aprendizaje

  • Comprender cómo JavaScript manipula el DOM en tiempo real.
  • Aplicar métodos básicos: createElement, appendChild, textContent, classList, setAttribute.
  • Recorrer arrays de datos para generar contenido dinámico.
  • Asociar eventos a elementos creados desde JS (addEventListener).
  • Separar correctamente HTML, CSS y JavaScript.

📝 Instrucciones

  1. Trabaja dentro de la carpeta evaluacion 2. Contiene los siguientes archivos:
    • index.html → archivo base (NO se modifica el contenido HTML).
    • script.js → archivo donde debes trabajar.
    • styles.css → estilos ya provistos (NO modificar).
    • blog-original.html → archivo de referencia visual del blog a replicar.
    • evaluacion2.html → este documento (instrucciones y rúbrica).
  2. Tu objetivo es que index.html se vea idéntico a blog-original.html, pero generando todo el contenido desde script.js.
  3. No puedes agregar etiquetas HTML dentro de index.html (excepto el div#app ya existente).
  4. Debes completar las funciones crearHeader(), crearArticulos(), crearSidebar() y crearFooter().
  5. El botón "Leer más" debe mostrar un alert() con el título del post correspondiente.
  6. Sube el resultado al repositorio o entrégalo comprimido en .zip con tu nombre.
